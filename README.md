# Arduino
This repository contains a collection of  circuit diagrams, documentation and mini/ big projects for building and testing embedded systems using the Arduino platform.

# About this reposority
**Basic projects**
- Learn how to code and controlling LEDs with and without buttons
- Types of 7-segment LED control such as: Light each segment; count from 0 to 9/ 9 to 0; with buttons
- Learn how to display on screen LCD1602 + module I2C
- Using basic components such as: servo motor, keypad, ...
