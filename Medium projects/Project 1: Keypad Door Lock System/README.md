# Components needed
- Arduino UNO R3
- LCD1602
- Module I2C for LCD1602
- Keypad 4x4
- Power 4.5V or 5V DC
- 1 Servo motor
- 1 Buzzer
- Mini testboard
  
# Ý tưởng
- Ấn nút nhấn để nhập mật khẩu -> Màn hình LCD thể hiện nơi nhập mật khẩu
  - Nhập đúng thì động cơ servo mở khóa
  - Nhập sai thì còi báo động 1 lần -> Tiếp tục nhập sai 3 lần thì còi báo động
